# Streamlit Dashboards

Welcome to my **Streamlit Dashboards repository**!

This repository contains interactive dashboards built with **Python and Streamlit** to explore and analyze financial data. Each dashboard is designed to provide a clean, user-friendly interface with insightful visualizations, summary metrics, and dynamic filters.

---

## Dashboards Included

### IDLC Finance PLC Stock Dashboard

Explore IDLC stock data from January 23, 2024, onward:

- **Close price and moving averages** (MA20, MA50, MA200)  
- **Daily and cumulative returns**  
- **Trading volume and 30-day rolling volatility**  
- **Positive vs negative daily returns pie chart**  
- **Summary metrics with corresponding dates**  
- **Sidebar controls to filter by date**

*(More dashboards may be added in the future.)*
